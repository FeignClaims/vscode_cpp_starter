========================================================================================================================
安装方法
========================================================================================================================

.. note::

  以下两步, 如果你的电脑上没有显示文件后缀, 请均尝试一下.

解压缩
  如果文件压缩为 **.zip / .7z / .rar** 文件，请 :KBD:`右键` 该文件, 然后选择 :KBD:`提取` 以提取文件. 这是系统内置的解压缩方法, 如果有其他压缩软件请随意.

安装
  :KBD:`右键` **.ttf** 或 **.ttc** 文件, 然后选择 :menuselection:`安装` 或 :menuselection:`为所有用户安装`.
