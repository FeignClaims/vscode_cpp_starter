更新日志#

仅对大型的更新进行记录, 便于读者了解是否需要更新和 如何更新.

2024/11/2#

软件更新

  • 将 llvm 更新至 19.

学习模板更新

  • 将 llvm 更新至 19.

2024/10/18#

软件更新

  • 请 linux 和 wsl 额外安装 libc++ 和 libc++abi: - ubuntu: sudo apt install libc++-dev libc++abi-dev - fedora: sudo dnf install libcxx-devel libcxxabi-devel

学习模板更新

  • 强制 clang 在 linux 和 wsl 上使用 libc++ 而不是 libstdc++

2024/09/30#

学习模板更新

  • 大量禁用 clang-tidy 的检查, 减轻初学者负担.

2024/05/12#

软件更新

  • 将 llvm 更新至 18.

学习模板更新

  • 将 llvm 更新至 18.

  • 完全重写简化学习模板的结构, 并直接在学习模板内添加说明文件.

2024/02/29#

学习模板更新

  • 正式使用 CMake 进行代码管理.

  • 补充了利用 vcpkg 或 conan 简单添加第三方库的方式.

2024/02 ~ 2021/09#

以前的更新太杂, 且已没有提示如何更新的必要.
