安装 WSL#

以管理员身份打开 PowerShell#

按 Win⊞ + X, 然后选择 Windows PowerShell (管理员)(A) 从而以管理员身份打开 PowerShell.

备注

注意打开的窗口样式, 当输入命令后再次出现类似样式时, 说明命令执行完毕.

启用 WSL 功能#

警告

  • 一块一块地 复制 (点击代码右上角按钮)、粘贴 (窗口内右键! 如果习惯性按了 Ctrl + V, 狂按几次 删除键).

  • 多次 回车! 多次 回车! 多次 回车!

  • 等待执行完毕后继续下一行. (确定执行完毕的方法见上文.)

  • 如果实在卡住太长时间, Ctrl + C 终止执行, 重新输入并执行命令.

dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart

dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart

禁用 Hyper-V#

运行后按提示重启电脑.

Disable-WindowsOptionalFeature -Online -FeatureName Microsoft-Hyper-V-Hypervisor

更新 Linux 内核#

下载 Linux 内核更新包 并双击运行, 不断确认, 最终按 Finish 即可.

安装 WSL2#

下载 WSL 安装包 并双击 安装 即可.

安装完成后, 会自动弹出一个 WSL 的终端, 我们就在这个终端里进行之后的所有操作.

注册用户账号#

首次打开终端时, 会要求我们注册一个用户账号. 具体地, 我们需要输入用户名, 回车, 输入密码, 回车, 再次输入密码, 回车.
